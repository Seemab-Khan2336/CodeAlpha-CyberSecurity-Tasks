## 🕵️ Task 01: Network Sniffer
A powerful Python script that monitors and captures live network traffic. 
* **Functionality:** Extracts Source IP, Destination IP, and Protocol information from IP packets.
* **Use Case:** Network forensics, monitoring unauthorized data flow, and understanding packet structures.
* **File:** `task 01.py`

## 🔐 Task 02: Secure Login System
This task demonstrates the importance of data protection by evolving a basic login system into a secure one.
* **Phase 1 (Vulnerable):** `task 03.py` stores passwords in plaintext (highly insecure).
* **Phase 2 (Secure):** `task 03 (2).py` implements **SHA-256 Hashing**.
* **Key Feature:** Passwords are never stored directly; only their cryptographic hashes are compared during login.

## 🎣 Task 03: Phishing Awareness Training
A comprehensive presentation designed to educate users on modern cyber threats.
* **Content:** Identifying fake bank emails, recognizing urgent security alerts, and anatomy of a phishing attack.
* **Prevention Tips:** Enabling 2FA, URL verification, and sender authentication.
* **File:** `Phishing-Awareness-Training.pptx`
